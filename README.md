# usage(Linux-debian):
- Clone this repository
- Install follow library to run mp3 music
```
sudo apt-get install libsdl2-dev libsdl2-mixer-de
```
- Move to console-tetris folder
```
cd console-tetris
```
- Execute ./consoletetris
- Use full screen
